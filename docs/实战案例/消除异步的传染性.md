# 消除异步的传染性

## 什么是异步的传染性

异步的传染性指的是如果一个函数调用的函数是异步的，那么这个函数也是异步的，以此类推，这个异步函数上层调用链上的所有函数都是异步的。
看下面这个例子：

```javascript
async function getUser() {
    return await fetch('').then((resp) => resp.json())
}

async function m1() {
    return await getUser();
}

async function m2() {
    return await m1();
}

async function main() {
    return await m2();
}
```

对于普通的函数调用，这或许并不是很重要，但是在函数式编程中，这会导致所有纯函数因为异步而受到影响。
